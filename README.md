Este projeto demonstra a construção de um pipeline de dados completo (End-to-End), desde a extração e tratamento de dados brutos até a criação de um dashboard estratégico para tomada de decisão.

O objetivo foi analisar o desempenho de vendas da Amazon, identificando as categorias mais lucrativas e a correlação entre políticas de desconto e a satisfação do cliente.
